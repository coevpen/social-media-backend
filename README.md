# Social Media Backend

## Description

I was tasked to make the backend for a social media site using NoSQL/MongoDB. I created all of the CRUD routes and tested in Insomnia.

## Built With
* Nodejs
* JavaScript
* Expressjs
* MongoDB
* [Mongoose](https://mongoosejs.com/)
